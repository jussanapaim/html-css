Trabalhando com Listas

Listas Ordenadas

As ordered lists são aquelas que sua ordem é importante. Sua formatação padrão é:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

As ol têm dois parâmetros, o type (que APENAS podem ser '1', 'A', 'a', 'I' e 'i') e o start. Exemplificando:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Listas Desordenadas

As unordered lists são aquelas que sua ordem nãoé importante. Sua formatação padrão é:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

As ul têm umn parâmetros, o type (que APENAS podem 'disc', 'square' e 'circle'). Exemplificando:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Mistura de listas ol e ul

Apenas com ol

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4

Lista de Definições

As definition lists (dl) são como se fossem um dicionário onde temos os termos e suas definições, o termo é escrito na tag dt e a descrição na tag dd. Exemplo:

HTML
Linguagem de marcação para a criação do contéudo de um site.
CSS
Linguagem de marcação para a criação design do contéudo de um site.
JavaScript
Linguagem de programação para a criação de interatividade do site.
